Style Sheets




Structure of CSS
Declaration is a single instruction that defines the style of an element.
Selector - chooses the HTML element to style.
Property - the style attribute you want to modify.
Value - The setting of the attribute you want to apply to the element property.

A set of instructions to a Web browser on how to display various elements on a Web page is known as a style sheet. Every CSS, whether it is contained in a .css file, or embedded in the head element of an HTML document, is a series of instructions called statements.



Three Kinds of CSS




External Style Sheets

It allows you to develop a consistent style across pages and change the layout of your entire website by simply changing the external style sheet. It uses a separate css file and by linking it to an HTML file.
< link rel="stylesheet" href="sample.css" type="text/css">

Internal Style Sheets

Embedded styles or Internal style sheets are defined by the container tag <style></style> and placed within the head part of the HTML file. It provides style to the whole HTML file

Inline Style Sheets

We can define the style for a single element using the style attribute. There are various tags that have the style attribute that would represent its value.



External can't overwrite other style sheets. Internal can overwrite external. Inline overwrites both external and internal style sheets.



In HTML, comments are defined using <!-- --> while CSS comments are defined using /* */.
