Intro to CSS



CSS
Cascading Style Sheets describes how HTML elements are to be displayed in the website. It is the language we use to style a web page and to separate content from design. It can control the layout of multiple websites. It provides consistency and saves time. CSS helps update pages, position and layer obejects, and create custom tags. CSS saves typing, downloads and loads faster, have multiple links in one document, and is flexible. But you need to learn HTML and XHTML before learning CSS.



Layers of a Webpage


Each layer of the webpage has its specific use or function in the web page. Each plays an important role to be able to attain the goal or objective of a particular website.



Content Layer

This layer is where you define and layout the text, images, animation, sound, video and everything you want to put in a web page with the use of tag.

Presentation Layer

The presentation layer is where you can define how people see your web page in a browser.

Behavior Layer

The behavior layer is where you have a
real-time user interaction with the web page. The interaction may be simple validation of input or filling out a form or even a big as
web-based programs.
